# PyRamen
![](https://assets.epicurious.com/photos/54ad70036529d92b2c046e6b/6:4/w_620%2Ch_413/51187270_shoyu-ramen_1x1.jpg)

Python script to analyze how well a business did on a per_product basis in order to better understand which products are doing well, which are doing poorly, and, ultimately, which products may need to be removed or changed.
